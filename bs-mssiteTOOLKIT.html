Calculator Tools

Five powerful tools, one file

Each tool is accessible via a tab in the navigation bar. Results animate in on calculation and can be recalculated at any time.

01
Tab 1
GPA Calculator

Calculate semester GPA from any number of courses, then optionally compute a running cumulative GPA by entering prior credits and GPA. Supports three grading scales.

  • Add unlimited courses with custom names, credit hours, and letter grades
  • Switch between 4.0, 5.0, and 10-point grading scales on the fly
  • Enter previous cumulative GPA and credits to get updated overall GPA
  • Academic standing indicator: Summa Cum Laude, Magna Cum Laude, Cum Laude, Good Standing, Satisfactory, Academic Warning
  • Shows semester credits, relative score percentage, and total credits
02
Tab 2
Credit Hour Tracker

Monitor degree completion progress across all credit categories with visual progress bars and per-category breakdowns.

  • Choose from BS Math, MS Math, BS Applied, or MS Applied programs
  • Enter credits in 6 categories: Core, Elective, General Ed, Research, Lab/Seminar, Transfer
  • Overall percentage progress bar toward degree total
  • Per-category colored progress bars vs. required amounts
  • Eligibility notice when all requirements are met
  • Remaining credits calculation and warning when short
03
Tab 3
Statistics Tools

Three statistical calculators: descriptive statistics, probability distributions, and matrix operations — covering common needs in math coursework.

  • Descriptive stats: Mean, Median, Std Dev, Variance, Min, Max, Range, Q1, Q3, IQR, CV%, Count
  • Normal distribution: Z-score, P(X≤x), P(X>x)
  • Binomial distribution: P(X=k | n, p)
  • Poisson distribution: P(X=k | λ)
  • Matrix operations: 2×2 and 3×3 determinant, trace, invertibility check
  • Formulas displayed alongside every result
04
Tab 4
Grade Converter

Convert any percentage score to letter grade and GPA points under three international grading systems, with full reference tables for each.

  • Enter percentage (0–100) and select grading system
  • US Standard: A/B/C/D/F with +/- grades on 4.0 scale
  • Pakistan HEC: A+/A/B+/B/C+/C/D/F on 4.0 scale with descriptors
  • UK System: First Class / Upper Second / Lower Second / Third / Fail
  • Color-coded result badges: gold (≥80%), blue (≥65%), green (≥50%), red (below)
  • Complete reference tables viewable for all three systems
05
Tab 5
Thesis Progress

Track MS thesis completion by monitoring credit hours and research milestones, with a dynamic progress overview and status indicator.

  • Enter total thesis credits required and credits completed so far
  • Record program start semester for reference
  • 9 pre-loaded default milestones covering the full thesis journey
  • Check off milestones, edit names, remove, or add custom ones
  • Dual progress bars: credit progress + milestone completion %
  • Status indicator: Early Stage / In Progress / On Track 🎓

Statistics Tools

Descriptive stats, probability & matrices

Three statistical calculators bundled into one tab, covering the most common computation needs for mathematics coursework.

σ
Descriptive Statistics
  • Count (n)
  • Mean (average)
  • Median (middle value)
  • Standard Deviation (σ)
  • Variance (σ²)
  • Minimum and Maximum
  • Range (Max − Min)
  • Q1 (25th percentile)
  • Q3 (75th percentile)
  • IQR (Q3 − Q1)
  • Coefficient of Variation %
  • Input: comma or space separated values
P(x)
Probability Calculator
  • Normal distribution — Z-score, P(X≤x), P(X>x)
  • Binomial — P(X=k | n, p)
  • Poisson — P(X=k | λ)
  • Enter mean μ, std dev σ, and x value
  • Results displayed with full formula context
Normal: Z = (x − μ) / σ
Binomial: P(X=k) = C(n,k) · pᵏ · (1−p)ⁿ⁻ᵏ
Poisson: P(X=k) = λᵏ · e⁻λ / k!
[A]
Matrix Operations
  • Input a 2×2 or 3×3 matrix
  • Determinant calculation
  • Trace (sum of diagonal)
  • Invertibility check (det ≠ 0)
  • Rank estimate
  • Results displayed with formula
det(A₂) = ad − bc
det(A₃) = a(ei−fh) − b(di−fg) + c(dh−eg)
tr(A) = Σ aᵢᵢ

Visual Effects

Atmospheric depth is created through layered effects that don't rely on heavy CSS. The result is a polished look that loads fast.

  • Fixed grid pattern (40×40px) of faint blue lines across the full viewport
  • Two blur-filter glow orbs: gold top-right, blue bottom-left
  • All result panels fade in with a translateY animation on calculate
  • Tab switch triggers a fade + slide-up animation
  • Card hover raises border color toward gold tint
  • Input focus ring: 3px gold glow shadow

Technical Architecture

Designed for simplicity and portability. Works in any modern browser without an internet connection once loaded.

  • Single HTML file — no external JS libraries
  • Google Fonts loaded via CDN link (optional)
  • All state managed in plain JS arrays and variables
  • CSS custom properties for consistent theming
  • Responsive via CSS Grid with auto-fit columns
  • Mobile breakpoint at 700px — tabs stack, grids collapse
  • All calculations run instantly in-browser, no server needed
